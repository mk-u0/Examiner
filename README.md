# Examiner++

### A desktop app for administering exams

Build requirements:
- CMake
- C++ Compiler
- wxWidgets
- SQLite3
- nlohmann json
- OpenSSL

For msys2 (UCRT recommended) users on Windows run the following command:
```
pacman -S mingw-w64-ucrt-x86_64-toolchain mingw-w64-ucrt-x86_64-cmake mingw-w64-ucrt-x86_64-sqlite3 mingw-w64-ucrt-x86_64-wxwidgets3.2-msw mingw-w64-ucrt-x86_64-ninja mingw-w64-ucrt-x86_64-openssl mingw-w64-ucrt-x86_64-nlohmann-json
```
